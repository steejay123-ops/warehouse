# پروژه مدیریت انبار (Warehouse Management System)

این یک سیستم یکپارچه برای مدیریت، انبارداری، صدور اسناد و شمارش موجودی است. پروژه از دو بخش مجزا تشکیل شده است:
- **بک‌اند (Backend):** نوشته شده با فریم‌ورک `Django` پایتون
- **فرانت‌اند (Frontend):** نوشته شده با فریم‌ورک `Angular` و تایپ‌اسکریپت

---

## راهنمای نصب و راه‌اندازی برای توسعه‌دهندگان

برای راه‌اندازی محیط توسعه (Development Environment) روی سیستم شخصی خود (در محیط‌هایی مثل VSCode)، لطفاً مراحل زیر را به دقت دنبال کنید.

### پیشنیازها
پیش از شروع، مطمئن شوید نرم‌افزارهای زیر روی سیستم شما نصب هستند. در صورتی که این ابزارها را ندارید، از لینک‌ها و دستورات زیر برای نصب آن‌ها استفاده کنید:

1. **Python (نسخه ۳.۸ به بالا)**
   - **ویندوز/مک:** به [سایت رسمی Python (python.org)](https://www.python.org/downloads/) مراجعه کرده و آخرین نسخه را دانلود و نصب کنید. (دقت کنید در مراحل نصب ویندوز، تیک گزینه `Add Python to PATH` را حتماً بزنید).
   - **لینوکس (اوبونتو):** `sudo apt update && sudo apt install python3 python3-venv python3-pip`

2. **Node.js و npm**
   - **ویندوز/مک:** به [سایت رسمی Node.js (nodejs.org)](https://nodejs.org/) مراجعه کرده و نسخه `LTS` را دانلود و نصب کنید. (npm به همراه Node.js نصب می‌شود).
   - **لینوکس (اوبونتو):** `sudo apt update && sudo apt install nodejs npm`

3. **Git**
   - **ویندوز:** برنامه [Git for Windows](https://gitforwindows.org/) را دانلود و نصب کنید.
   - **مک:** از طریق ترمینال با دستور `git --version` (در صورت نصب نبودن، ابزار خط فرمان مک به صورت خودکار پیشنهاد نصب می‌دهد) یا از طریق `brew install git`.
   - **لینوکس (اوبونتو):** `sudo apt update && sudo apt install git`

### بخش اول: راه‌اندازی بک‌اند (Backend)

یک ترمینال (Terminal) جدید باز کنید و وارد پوشه بک‌اند شوید:
```bash
cd warehouse-backend
```

1. **ساخت محیط مجازی (Virtual Environment):**
   ```bash
   python -m venv venv
   ```

2. **فعال‌سازی محیط مجازی:**
   - در ویندوز: `.\venv\Scripts\activate`
   - در مک/لینوکس: `source venv/bin/activate`

3. **نصب پکیج‌ها و کتابخانه‌های پایتون:**
   ```bash
   pip install -r requirements.txt
   ```

4. **پیکربندی اولیه و ساخت دیتابیس:**
   برای ایجاد دیتابیس، اعمال مایگریشن‌ها، ساخت کاربران پیش‌فرض و انبار نمونه، فقط کافیست دستور آماده‌ی زیر را اجرا کنید:
   ```bash
   python manage.py setup_project
   ```
   *(این دستور تمامی نقش‌های سیستمی و کاربران تستی را به صورت خودکار ایجاد می‌کند).*

5. **اجرای سرور بک‌اند:**
   ```bash
   python manage.py runserver 8000
   ```
   > سرور پایتون در آدرس `http://127.0.0.1:8000` اجرا خواهد شد. این ترمینال را نبندید.

---

### بخش دوم: راه‌اندازی فرانت‌اند (Frontend)

یک ترمینال **جدید** باز کنید (تا سرور بک‌اند بسته نشود) و وارد پوشه فرانت‌اند شوید:
```bash
cd warehouse-front
```

1. **نصب پکیج‌های Node.js:**
   ```bash
   npm install
   ```

2. **اجرای سرور انگولار (Angular):**
   ```bash
   npm run start
   ```
   > سرور فرانت‌اند به صورت پیش‌فرض در آدرس `http://localhost:4200` اجرا خواهد شد.

## اطلاعات ورود به سیستم (Credentials)

پس از بالا آمدن هر دو سرور، مرورگر خود را باز کرده و به آدرس `http://localhost:4200` بروید. 
به لطف اجرای دستور `setup_project` در بک‌اند، شما می‌توانید با هر یک از کاربران تستی زیر وارد سیستم شوید و فرآیندهای مختلف را بررسی کنید:

| نام کاربری (Username) | رمز عبور (Password) | نقش در سیستم |
| :--- | :--- | :--- |
| **admin** | `admin` | مدیر کل سیستم (Superuser) |
| **sysadmin** | `123456` | مدیر سیستم (System Admin) |
| **whmanager** | `123456` | سرپرست انبار (Warehouse Manager) |
| **clerk** | `123456` | انباردار (Stock Clerk) |
| **supervisor** | `123456` | سرپرست شمارش (Count Supervisor) |
| **counter** | `123456` | شمارشگر (Counter) |
| **manager** | `123456` | مدیر پروژه (Project Manager) |
| **exporter** | `123456` | اپراتور صدور (Exporter) |

## ساختار پروژه
- **`warehouse-backend/`**: حاوی اپلیکیشن‌های جنگو (مانند `accounts`، `inventory`، `warehouses` و...).
- **`warehouse-front/`**: حاوی کدهای انگولار (کامپوننت‌ها، سرویس‌ها، مدل‌ها و لاگیک کاربری).
- **`Documents/`**: شامل مستندات داخلی، فرآیندها و تاریخچه تصمیم‌گیری‌های پیاده‌سازی.
